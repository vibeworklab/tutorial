## 设计要点说明

| 区域 | 设计考量 |
|------|----------|
| 顶部导航 | 固定，核心功能入口 |
| 搜索栏 | 首页突出，方便快速查找 |
| 分类标签 | 常见分类，减少筛选步骤 |
| 物品卡片 | 图片+价格+位置，关键信息一目了然 |
| 详情页 | 大图+价格+卖家信息+联系方式 |
| 底栏 | 常用功能，发布按钮居中突出 |
