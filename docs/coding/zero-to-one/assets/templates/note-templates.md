# 学霸笔记模板配置

定义每课笔记的内容结构和布局要求，供 agent 生成时参考。

---

## 通用规范

### 尺寸
- viewBox: `800 x 1000`
- 边距: 30px
- 圆角: 8px

### 颜色系统
```css
--primary: #FF6B35;      /* 橙色标题 */
--success: #4CAF50;      /* 绿色正面 */
--danger: #EF5350;       /* 红色负面 */
--warning: #FFC107;      /* 黄色高亮 */
--info: #2196F3;         /* 蓝色信息 */
--purple: #9C27B0;       /* 紫色特殊 */
```

### 字体大小
```css
.title: 26px bold
.subtitle: 14-15px medium
.text: 11-13px regular
.small: 10px
```
