# 第15课：推广物料——让AI帮你写文案做海报

![](./assets/notes/L15-notes.svg)

## 场景引入

你的校园二手平台已经上线了！

你激动地发了条朋友圈：
> "我做的校园二手平台上架啦！快来看看！"
> + 一个链接

然后你刷新了一下...没有人点赞。
再刷新...还是没有。

**为什么？**

- 朋友圈文案太平淡，没吸引到人
- 没有配图，视觉冲击力不够
- 没说清楚这个平台能干嘛
- 没有引导用户怎么用

这一课，我们来做**推广物料**——让AI帮你写吸引人的文案、做好看的海报。

---

## 思考过程

### 什么是推广物料？

推广物料就是你用来"推销"产品的所有东西：

```
推广物料 = 文案 + 图片 + 视频 + 教程
              ↓       ↓       ↓       ↓
           朋友圈   海报    短视频   使用指南
           公众号   Banner  演示    FAQ
```

类比一下：
- 产品是"商品"
- 推广物料是"包装"和"广告"
- 再好的商品，包装不好也没人买

### 推广的核心：对谁说、说什么、怎么说

在让AI写文案之前，你要想清楚三个问题：

**1. 对谁说？（目标用户）**
- 本校学生
- 有闲置物品的人
- 想淘便宜货的人

**2. 说什么？（核心价值）**
- 能卖闲置换钱
- 能淘到便宜好货
- 校园内交易，安全方便

**3. 怎么说？（渠道风格）**
- 朋友圈：短小精悍、口语化
- 公众号：详细、有故事
- 海报：视觉冲击、重点突出

**【人类决策点】** AI可以帮你写文案，但它不知道你的用户是谁、你的平台有什么特色。这些信息你需要先告诉AI。

## 知识提炼

### 文案写作技巧

**1. AIDA模型**

```
A - Attention（注意）：用标题或开头抓住眼球
I - Interest（兴趣）：描述和用户相关的痛点
D - Desire（欲望）：展示你的解决方案和价值
A - Action（行动）：告诉用户下一步做什么
```

比如校淘的文案：
- Attention：毕业季！带不走的不是青春...
- Interest：扔了可惜，带走麻烦
- Desire：校园卡认证，本校圈子，安全交易
- Action：扫码发布你的第一个闲置

**2. 针对不同用户写不同文案**

| 用户类型 | 关注点 | 文案风格 |
|----------|--------|----------|
| 卖家 | 能不能卖掉？快不快？ | 强调效率、曝光 |
| 买家 | 有没有好东西？靠谱吗？ | 强调安全、便宜 |
| 围观群众 | 这是啥？跟我有关系吗？ | 用故事、用幽默 |

**3. 真实感很重要**

```
❌ "最好用的校园二手平台"（太空泛）
✅ "昨天室友卖掉了用不到的台灯，30块"（有细节）
```

### 海报设计原则

**1. 一眼看清**

海报只有3秒的注意力时间。主信息要在3秒内被理解。

```
核心信息层次：
第1眼看到：标题（毕业带不走？校淘帮你卖）
第2眼看到：价值点（本校圈子·安全交易）
第3眼看到：行动（扫码）
```

**2. 视觉统一**

- 颜色不超过3种
- 字体不超过2种
- 风格和产品调性一致

**3. 留白**

不要塞满！留白让海报"呼吸"，信息更清晰。

### 推广渠道选择

| 渠道 | 特点 | 适合内容 |
|------|------|----------|
| 朋友圈 | 熟人传播，信任度高 | 短文案+链接/海报 |
| 班级群 | 精准触达 | 简洁介绍+链接 |
| 公众号 | 长内容，可被搜索 | 详细教程、故事 |
| 小红书 | 年轻人多，容易爆 | 图文教程、真实分享 |
| 抖音 | 流量大，竞争大 | 短视频、趣味内容 |

### 文案模板库

**朋友圈文案公式**：
```
痛点/共鸣 + 解决方案 + 行动号召
例：宿舍东西堆不下？（痛点）
    上校淘一键发布（方案）
    扫码试试→（行动）
```

**海报文案公式**：
```
主标题（1句话抓住注意）
副标题（说清楚是什么）
3个卖点（为什么选你）
行动号召（扫码/点击）
```
